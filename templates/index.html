Overview

Wisp is a lightweight API that allows you to fetch profile pictures from various services. It provides a unified interface for retrieving avatars with built-in caching.

Quick Start

Get a profile picture:

GET /{service}/{username}
i
Will return a fallback image incase of an error.
Example: GET /github/octocat

Batch request:

GET /batch?github=octocat&twitter=twitter
